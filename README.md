# Analysis code for: HaemaTONIC: Human Haematopoiesis using cyTokines and Optimised Niches In Culture

## Description
This codes were used to make plots for thsi resource papaers fo all Figures.

## Data availability
Raw data and count matrices are available at GEO: GSExxxxxx
(https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSExxxxxx)

## Software requirements
- R version 4.5.1
- Seurat v5.5.0


## How to run
1. Download count matrices from GEO (GSExxxxxx)
2. Update file paths in `00_config.R`
3. Run scripts in numbered order
